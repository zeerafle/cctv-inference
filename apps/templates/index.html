CCTV Accident Inference API

There are over 52 CCTV. Visit the appropriate inference API endpoint for each CCTV by adding the /prediction suffix to the last identifier path in the CCTV URL provided in this link.

For example, the CCTV URL for Simpang Antasari Siradj Salman is https://diskominfo.samarindakota.go.id/api/cctv/simpang-antasari-siradj-salman.

To access the inference API, go to `localhost/prediction/simpang-antasari-siradj-salman`
